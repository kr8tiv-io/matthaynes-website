# Matt Haynes | Personal Brand & Link Directory

Welcome to the central hub and operating structure for Matt Haynes—Founder, Designer, and Operator building across AI, automation, and blockchain.

This repository contains the source code for Matt's personal brand portfolio and high-utility Link-in-bio page, architected for a premium, agency-level aesthetic with seamless motion design and a 3D structural core.

## 🪐 Architecture & Design
The site was built as an **Awwwards-worthy** showcase, bridging deep technical infrastructure with uncompromising design.

- **GSAP Driven Motion:** Cinematic vertical scroll experiences utilizing GreenSock Animation Platform to govern multi-layered HTML5 background video cross-fading, giving distinct sections isolated breathing room.
- **Three.js WebGL Foundation:** A continuous, low-friction WebGL layer rendering an interactive constellation of dynamic particles responding natively to cursor velocity constraints and Lenis scrolling speed.
- **Micro-Glassmorphism:** Ethereal, translucent UI systems. Floating system cards, stark typography scaling, and deeply crushed drop-shadow properties to guarantee clean legibility against active video backgrounds without heavy boxy visual disruptions.
- **Typography:** Strict architectural deployment of Google Fonts' `Sora` and `Manrope`, alongside the `@font-face` import of the sharp, technical `NeueMachina` typeface.

## 🚀 Running Locally
Built on raw native specs (HTML5, CSS3, Vanilla JS) for optimal control. To initialize the site locally without caching blocks:
```bash
npx serve
```
Navigate to `http://localhost:3000` to view the live environment.
